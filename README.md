# Blockchain-Based Farm Marketplace

A cross-platform mobile application connecting Farmers and Buyers directly using Blockchain for secure transactions and product ownership verification.

## 🚀 Tech Stack

### Frontend
- React Native (Expo)
- TypeScript
- Expo Router
- AsyncStorage

### Backend (Planned)
- Node.js
- Express.js
- MongoDB
- JWT Authentication

### Blockchain (Planned)
- Ethereum
- Solidity
- Hardhat
- Ethers.js

## 📱 Features

### Authentication
- ✅ Multi-role login (Farmer, Buyer, Admin)
- ✅ Registration for Farmer and Buyer
- ✅ Local storage for user credentials
- ✅ Password visibility toggle
- ✅ Form validation
- ✅ Remember Me functionality
- ✅ Forgot Password (Coming soon)

### Dashboards
- ✅ Farmer Dashboard with statistics
- ✅ Buyer Dashboard with statistics
- ✅ Admin Dashboard with management options
